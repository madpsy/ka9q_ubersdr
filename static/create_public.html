Welcome to the Public Instance Registry

📡 What is Instance Reporting?

Instance reporting allows your UberSDR to be listed in the public registry at ubersdr.org, making it discoverable by other radio enthusiasts worldwide.

✅ Benefits

  • Share your SDR with the global amateur radio community
  • Contribute to propagation studies and band condition monitoring
  • Help others learn about SDR technology
  • Join a network of public SDR receivers

🔒 Privacy & Security

  • Only basic information is shared (callsign, location, connection details)
  • Your admin password is never transmitted
  • You can disable reporting at any time
  • Connection details are optional - you control what's shared

📊 What Gets Reported?

  • Instance UUID (unique identifier)
  • Callsign and location (from your admin settings)
  • Public connection details (if you provide them)
  • Basic statistics (number of users, uptime)

How to Use

1. Tune to a CW signal: Use CWU or CWL mode with narrow bandwidth (¬±250 Hz recommended)

2. Adjust settings: Set target frequency to match the CW tone (typically 600-800 Hz)

3. Decoder adapts: Speed (WPM) is automatically detected from the signal timing

4. Signal threshold: Adjust if decoder is too sensitive or not detecting signals
